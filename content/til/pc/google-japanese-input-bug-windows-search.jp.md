Google日本語入力には、Windows検索が正しく動作しなくなるバグがありました。検索バーにテキストを入力できません。

> 環境：<br> - Windows 11（build 22631.4169）<br>- Google日本語入力（GoogleJapaneseInput-2.30.5590.0+24.11.9）

バグ修正版がリリースされました。

> Windows Stable: 2.30.5620.0（1%）/ 2.29.5370.0（99%）<br>
> Windows Develop: 2.30.5623.100（2.30.5618.100から更新）<br>以下のURLから安定版をダウンロードできます：<br>
> https://dl.google.com/release2/misc/adgy2mhczyv34wo6gkbhvii3lkuq_2.30.5620.0/GoogleJapaneseInput64-2.30.5620.0.msi
