## VS CodeのGitHub Copilotでプロンプトにワイルドカード`*`を使う

GitHub Copilotのプロンプトでワイルドカード文字`*`を使用できます。

この強力な機能により、Copilotにより柔軟で一般化された指示を与えることができ、コード提案を生成する際に複数のファイルやパターンを考慮することを理解させることができます。

例:

```text
`src/*.py` を解析してください。
```
