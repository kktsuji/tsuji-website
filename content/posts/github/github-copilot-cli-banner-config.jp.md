## GitHub Copilot CLIのバナー設定

1. `~/.copilot/config.json`ファイルを開く
2. `"banner": "always"`を設定する:

```json
{
  "banner": "always",
  "render_markdown": true,
  "model": "claude-sonnet-4.6",
  "reasoning_effort": "medium"
}
```
