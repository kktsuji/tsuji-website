## Rules Settings

You can set up automatic code review by Copilot for GitHub Pull Request in the Rules settings of your GitHub repository. This feature allows you to automatically request a code review from Copilot when a pull request is created or updated, which can help maintain code quality and assist with code review.

- Go to GitHub repository > Settings > Rules > New ruleset
- Enable `Automatically request Copilot code review`
