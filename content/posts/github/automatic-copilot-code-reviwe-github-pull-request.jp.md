## Rulesの設定

GitHubリポジトリのRules設定で、GitHub Pull RequestのCopilotによる自動コードレビューを設定できます。この機能により、pull requestが作成または更新された際にCopilotへのコードレビューを自動でリクエストでき、コード品質の維持やコードレビューの支援に役立ちます。

- GitHubリポジトリ > Settings > Rules > New rulesetに移動
- `Automatically request Copilot code review`を有効化
