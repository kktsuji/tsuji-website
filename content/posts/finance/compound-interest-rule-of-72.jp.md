72の法則は、固定金利が与えられた場合、投資の価値が2倍になるまでにどれだけの時間がかかるかを計算するシンプルな公式である。

$ y = i \div r $

ここで$y、i、r$はそれぞれ予想期間[年]、投資額[任意の通貨]、固定年利率[%]を表す。

$100を投資した場合、計算結果は以下の通りである:

| 金利[%] \ 年 | 0   | 1.5 | 3   | 6   | 12  | 24  | 48  | 72  |
| ------------ | --- | --- | --- | --- | --- | --- | --- | --- |
| 0.0          | 100 |     |     |     |     |     |     | 100 |
| 1.0          | 100 |     |     |     |     |     |     | 200 |
| 1.5          | 100 |     |     |     |     |     | 200 |     |
| 3.0          | 100 |     |     |     |     | 200 |     |     |
| 6.0          | 100 |     |     |     | 200 |     |     |     |
| 12.0         | 100 |     |     | 200 |     |     |     |     |
| 24.0         | 100 |     | 200 |     |     |     |     |     |
| 48.0         | 100 | 200 |     |     |     |     |     |     |
