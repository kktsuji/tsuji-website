## VS CodeでGitHub CopilotのChat Debug Viewを表示する

1. Visual Studio Codeを開く
2. Chatを開く（Ctrl+Alt+IまたはトップメニューバーのToggle Chatアイコンをクリック）
3. Chatパネルの右上にある「...」メニューをクリックする
4. ドロップダウンメニューから「Show Chat Debug View」を選択する

GitHub Copilotとのチャットインタラクションに関する詳細なログと情報を確認できます。これは問題のデバッグやCopilotがどのように提案を生成するかを理解するのに役立ちます。
